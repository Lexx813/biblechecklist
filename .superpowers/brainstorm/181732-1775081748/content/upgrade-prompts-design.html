What changes with upgrade prompts

  • Generic banner → contextual triggers — the same "Upgrade to Premium" banner on every page gets replaced with specific messages tied to what the user just did
  • 4 key trigger moments:
    • Book completion → push reading plans
    • 7-day streak → push streak protection
    • 3 quiz levels → push AI tools
    • Hitting a premium-gated feature → specific feature pitch, not generic
  • Each prompt has one CTA and one dismiss — no guilt, no dark patterns. "Not now" is always visible.
  • Home sidebar banner becomes rotating — cycles through different value props based on what the user has and hasn't tried yet, rather than showing the same message forever
  • Prompts are one-shot — once dismissed, that specific trigger doesn't fire again for 7 days
